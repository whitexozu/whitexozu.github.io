    Nginx란 Apache같은 웹 서버입니다.
    Ubuntu 16에 Eninx 설치 후 Spring boot으로 실행 된 8080 포트를 연결 해보겠습니다.
    방화벽이 없는 상태에서 진행 했습니다.

    목표

    • Ubuntu 16에 Eginx 설치
    • 8080 포트 eginx 연결
    • 옵션 설정
    • 권한 변경

    Nginx 설치

    설치

    sudo apt-get install nginx

    버전확인

    nginx -v
nginx version: nginx/1.10.3 (Ubuntu)

    기본 화면 접속

    http://localhost

    서비스 시작

    sudo service nginx status
sudo service nginx restart

    설정

    Nginx 설치 위치

    /etc/nginx

    설정 파일

    $ sudo vi /etc/nginx/nginx.cnf

    8008 포트 연결

    $ sudo vi /etc/nginx/conf.d/tomcat.conf

    upstream tomcat {
    ip_hash;
    server 127.0.0.1:8080;
}

server {
    listen       80;
    server_name  localhost;

    access_log   /var/log/nginx/tomcat_access.log;

    location / {
        proxy_set_header        Host $http_host;
        proxy_set_header        X-Real-IP $remote_addr;
        proxy_set_header        X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header        X-Forwarded-Proto $scheme;
        proxy_set_header        X-NginX-Proxy true;

        proxy_pass http://localhost:8080;
        proxy_redirect          off;
        charset utf-8;
    }
}

    기본 설정 제거

    • nginx.conf의 include /etc/nginx/sites-enabled/*; 를 주석처리 해서 default 설정을 꺼준다.

    nginx reload

    nginx -s reload
