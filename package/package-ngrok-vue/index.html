    [ngrok] Vue 핫리로딩을 외부 단말에서

    On This Page

    • ngrok 설치
    • vue.config.js 수정
    • 실행
    • ngork 실행
    • 모바일에서 접속

    Vue로 Frontend를 개발하면서 편리한점중 하나는 수정한 내용을 바로바로 확인 할 수 있는 핫리로딩 입니다.
    그리고 핫리로딩을 모바일 기기같은 외부 단말 에서 사용할수 있게 해주는 package가 있습니다.
    바로 ngrok 입니다.
    한번 사용해보겠습니다.

    ngrok 설치

    $ npm install -g ngrok

    window는 ngrok 사이트에서 ngrok.exe 파일을 받아서 실행해야 합니다

    vue.config.js 수정

    module.exports = {
    ...생략...
    devServer: {
        disableHostCheck: true
    }
}

    실행

    $ npm run serve

    ngork 실행

    $ ngrok http 8080 #실행 포트 입력
